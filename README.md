# KillSay Mod

> 击杀玩家后自动发送自定义消息的 Fabric 客户端模组 (1.21.4)

---

## 功能

- 击杀玩家后自动从预设词汇中随机选取一条发送到聊天栏
- 支持 `@` 提及被击杀者
- 占位符替换：玩家名、坐标、血量、手持物品等
- 多词汇文件支持，游戏内热切换
- 测试模式：可在游戏中预览占位符效果
- 按键绑定开关

---

## 占位符

| 占位符 | 说明 |
|--------|------|
| `{name}` / `@{name}` | 被击杀者游戏名 |
| `{killer}` / `@{killer}` | 击杀者游戏名 |
| `{health}` | 击杀者当前生命值（整数） |
| `{item}` / `{weapon}` | 击杀者手持物品名称 |
| `{x}` `{y}` `{z}` | 击杀者坐标 |
| `{random}` | 6 位随机数字 |
| `{randomletters}` | 6 位随机大小写字母 |

---

## 指令

| 指令 | 说明 |
|------|------|
| `/ks` | 开关自动喊话 |
| `/ks refresh` | 重新加载当前词汇文件 |
| `/ks reset` | 恢复默认词汇 |
| `/ks test` | 开关测试模式（每 0.5 秒发一条） |
| `/ks list` | 列出所有可用词汇文件 |
| `/ks load <文件名>` | 加载指定词汇文件 |
| `/ks delete <文件名>` | 删除词汇文件 |
| `/ks add <文件名>` | 创建新词汇文件 |
| `/ks open <文件名>` | 用系统编辑器打开词汇文件 |

## 配置文件

所有文件位于游戏目录下的 `killsay/`：

```
.minecraft/
└── killsay/
    ├── README.md          ← 帮助文档
    ├── current            ← 当前词汇文件名（自动管理）
    ├── enabled            ← 开关状态（自动管理）
    ├── togglekey          ← 按键绑定键码
    └── vocabulary/
        ├── killsay.txt    ← 默认词汇文件
        └── ...
```

格式：每行一条消息，空行自动过滤。

## 击杀检测机制

1. **攻击跟踪**：攻击玩家时记录目标血量、位置
2. **Tick 检测**：每帧检查目标状态
   - 目标死亡 → 5 秒内攻击过 → 立即发送
   - 目标消失 → 加入待处理队列
3. **待处理队列**：目标消失 500ms 后确认世界中无同名存活玩家 → 发送
4. **Mixin 拦截**：`Entity.remove()` 触发时直接检测死亡

防误判：
- 单帧位移 > 10 格（珍珠瞬移等）→ 标记为传送 → 消失后不发送
- 目标距本地玩家 > 256 格 → 停止跟踪
- 攻击超过 8 秒 → 停止跟踪
