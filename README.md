## 占位符

{name} 被击杀者的游戏名
{killer} 你自己的游戏名
{health} 你当前的生命值（整数）
{item}/{weapon} 你手中持的物品名称
{x} {y} {z} 你的坐标
{random} 随机数 0-999999（6位数字）
{randomletters} 随机字母组合（固定6个大小写字母）


# 指令说明

### /ks
开关自动喊话功能。再次输入切换开关状态。

### /ksload
重新加载当前词汇文件。修改了词汇文件后在游戏中直接重载，无需重启游戏。

### /ksreset
恢复默认词汇。会将 killsay.txt 重置为初始内容并切回默认词汇。

### /kstest
开关测试模式。开启后每 0.5 秒自动发送一条当前词汇中的随机消息，用于测试占位符效果。再次输入关闭。

### /kswords list
列出 vocabulary/ 目录下所有可用的 .txt 词汇文件，当前加载的文件前会显示 * 标记。

### /kswords load <文件名>
加载指定的词汇文件（不需要输入 .txt 后缀，支持 Tab 补全）。加载后会自动保存当前词汇，下次启动游戏时自动加载同一个文件。

### /kswords delete <文件名>
删除指定的词汇文件（不能删除默认的 killsay.txt）。

## 配置文件

所有词汇文件存放在游戏目录下的 killsay/vocabulary/ 文件夹中：

.minecraft/
└── killsay/
    ├─── README.md       ← 帮助文档
    ├─── current          ← 记录当前词汇文件名（自动管理）
    └─── vocabulary/
        ├─── killsay.txt    ← 默认词汇文件
        ├─── myphrases.txt  ← 你可以新建自己的词汇文件
        └─── ...

文件格式：每行一条消息，/kswords load 自动过滤空行，加载所有非空行作为消息池。

注意：切换词汇后重启游戏会自动加载上次使用的文件，无需每次手动切换。
