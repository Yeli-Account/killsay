## 📋 占位符

| 占位符 | 支持 @ 前缀 | 说明 |
|--------|:----------:|------|
| `{name}` | `@{name}` | 被击杀者的游戏名 |
| `{killer}` | `@{killer}` | 你自己的游戏名 |
| `{health}` | ❌ | 你当前的生命值（整数） |
| `{item}` / `{weapon}` | ❌ | 你手中持的物品名称 |
| `{x}` `{y}` `{z}` | ❌ | 你的坐标 |
| `{random}` | ❌ | 随机数 0-999999（6位数字） |
| `{randomletters}` | ❌ | 随机字母组合（固定6个大小写字母） |

## 📖 指令说明

### `/ks`
开关自动喊话功能。再次输入切换开关状态。

### `/ksreload`
重新加载当前词汇文件。修改了 `killsay.txt` 后在游戏中直接重载，无需重启游戏。

### `/ksreset`
恢复默认词汇。会将 `killsay.txt` 重置为初始内容并重新加载。

### `/kstest`
开关测试模式。开启后每 0.5 秒自动发送一条当前词汇中的随机消息，用于测试占位符效果。再次输入关闭。

### `/kswords list`
列出 `killsay/` 目录下所有可用的 `.txt` 词汇文件，当前加载的文件前会显示 `*` 标记。

### `/kswords load <文件名>`
加载指定的词汇文件（不需要输入 `.txt` 后缀，自动补全）。  
示例：`/kswords load myphrases`

### `/kswords delete <文件名>`
删除指定的词汇文件（不能删除默认的 `killsay.txt`）。  
示例：`/kswords delete myphrases`

## ⚙️ 配置文件

所有词汇文件存放在游戏目录下的 `killsay/` 文件夹中：

```
.minecraft/
└── killsay/
    ├── killsay.txt      ← 默认词汇文件
    ├── myphrases.txt    ← 你可以新建自己的词汇文件
    └── ...
```

**文件格式：** 每行一条消息，`/kswords load` 自动过滤空行，加载所有非空行作为消息池。
